# stream-client

## ⚡️ Prerequisite
-   [Java](https://www.oracle.com/java/technologies/downloads/) (8)
-   [Maven](https://maven.apache.org/download.cgi)

## ⚙️ Build & Run
```bash
mvn spring-boot:run
```

## 🎯 Test
To test using HTTP Server use this script:
```bash
run_http.sh
```

To test using GRPC Server use this script:
```bash
run_grpc.sh
```
